1) Instalar node

2) Instalar pnpm =>  npm install -g pnpm

3) Verificar instalaciones

node -v
npm -v
pnpm -v   # si instalaste pnpm

4) Crearemos un Proyecto De Cero

	pnpm create next-app proyecto01 (ya tiene el nombre del proyecto)

NOTA: si por las dudas nos da error 

	Abrí PowerShell como Administrador.

	Ejecutá:

		Set-ExecutionPolicy RemoteSigned -Scope CurrentUser


	Te pedirá confirmación → escribí S y presioná Enter.



Probá de nuevo:

pnpm create next-app proyecto01
