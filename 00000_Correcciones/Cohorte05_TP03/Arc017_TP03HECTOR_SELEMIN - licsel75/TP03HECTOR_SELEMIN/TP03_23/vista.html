Ejercicio Nro. 23: Definir una arrow function que reciba una palabra y determine si contiene al menos dos letras “s” (mayúsculas o minúsculas). La función debe recorrer la palabra con un ciclo for, utilizando un contador o una bandera booleana. Debe devolver true si se cumplen las dos apariciones y false en caso contrario. No puede utilizar métodos como includes() o indexOf(). Nota: Debe devolver un boolean (true ó false).
