10) Diferentes formas de ejecutar los archivos .js

    node index.js

    npx nodemon index.js

11) Modificar el package json para reiniciar la aplicación

    "scripts": {
        "start": "node index.js",
        "dev": "nodemon index.js"
    }


    Diferencia entre npm run dev y npm run start:

        npm run dev: 
            Utiliza nodemon, que es útil para el desarrollo 
            porque vigila los cambios en el código y reinicia 
            automáticamente el servidor cuando detecta modificaciones.

        npm run start: 
            Utiliza node para iniciar la aplicación, 
            sin vigilancia de cambios. Es más adecuado para cuando tu código
            está listo y no esperas hacer modificaciones frecuentes.
