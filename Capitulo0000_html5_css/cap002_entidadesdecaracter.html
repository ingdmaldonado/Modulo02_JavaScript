Entidades de Caracteres en HTML

Este documento muestra cómo representar caracteres especiales usando entidades HTML sin que el navegador los interprete como etiquetas o símbolos.

Simbolizando Etiquetas HTML

Mostrar una etiqueta como texto: &lt;h1&gt;Hola Mundo&lt;/h1&gt;

Resultado: <h1>Hola Mundo</h1>

Entidades Comunes

  • &lt; = < (menor que)
  • &gt; = > (mayor que)
  • &amp; = & (ampersand)
  • &quot; = " (comillas dobles)
  • &apos; = ' (comilla simple)

Caracteres Acentuados y Especiales

  • &aacute; = á (á)
  • &eacute; = é (é)
  • &iacute; = í (í)
  • &oacute; = ó (ó)
  • &uacute; = ú (ú)
  • &ntilde; = ñ (ñ)
  • &uuml; = ü (ü)
  • &iexcl; = ¡ (¡)
  • &iquest; = ¿ (¿)

Entidades Matemáticas y Símbolos

  • &plusmn; = ± (±)
  • &times; = × (×)
  • &divide; = ÷ (÷)
  • &le; = ≤ (≤)
  • &ge; = ≥ (≥)
  • &ne; = ≠ (≠)
  • &frac12; = ½ (½)
  • &sum; = ∑ (∑)
  • &infin; = ∞ (∞)

Moneda y Tipografía

  • &euro; = €
  • &dollar; = $
  • &pound; = £
  • &yen; = ¥
  • &copy; = ©
  • &reg; = ®
  • &trade; = ™

Flechas y Otros

  • &rarr; = → (→)
  • &larr; = ← (←)
  • &uarr; = ↑ (↑)
  • &darr; = ↓ (↓)
  • &hearts; = ♥ (♥)
  • &spades; = ♠ (♠)
  • &clubs; = ♣ (♣)
  • &diams; = ♦ (♦)
