Buenas Prácticas

  • Usar alt en todas las imágenes por accesibilidad.
  • Usar target="_blank" con rel="noopener noreferrer" para seguridad.
  • Evitar enlaces vacíos (sin href).
  • Optimizar el tamaño de las imágenes para carga rápida.
